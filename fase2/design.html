Próposito del Sitio

Brindar una herramienta didáctica para el Aprendizaje de los temas relacionados al diseño de Páginas web tales como Flex Box, Listas y Enlaces.

Flex box

Modelo de diseño CSS3 hace que unas colecciones de elementos adaptables en un contenedor se organicen de manera automática en filas o columnas de acuerdo al tamaño de la pantalla del dispositivo, este método permite que los elementos de contenido se ensanchen para rellenar el espacio adicional y se encojan para caber en espacios más pequeños y así optimizar la capacidad de alineación.
En el flexbox existe un elementro pdre o principal que sostiene todos los elementos Flex y recibe el nombre de Flex Container y los elementos hijos directos se denomina Flex Item.

Listas

En el diseño de páginas web es posible crear listas mediante la utilización de etiquetas que pueden estar anidadas, es decir, una entre otras formando de esta manera árboles de diferente complejidad.

Imagen 2. Tipos de listas de HTML

Tipos de listas

Listas Desordenas (viñetas)

Las listas desordenadas muestran los ítems antecedidos por una viñeta y se utilizan la siguiente la etiqueta < u l >.

Ejemplo:

< u l >
< l i > Listas Ordenadas< / l i >
< l i > Listas Desordenadas< / l i >
< / u l >

En el navegador se visualizará de la siguiente manera:
• Listas Ordenadas
• Listas Desordenadas

Listas Ordenadas (Enumeradas)

Las listas Ordenadas muestran los ítems de manera enumerada en un solo nivel.

Ejemplo:

< o l >
< l i > Listas Desordenadas < / l i >
< l i > Listas Ordenas < / l i >
< / o l >

En el navegador se visualizará de la siguiente manera:
1. Listas Ordenadas
2. Listas Desordenadas

Listas de definiciones

En este tipo de listas se utilizan para destacar conceptos y definiciones con un texto indentado. Para este tipo de listas se utilizan las etiquetas < d t > y < d d > en lugar de < l i > .

Ejemplo:

< d l >
< d t > Listas de definiciones < / d t >
< d d > Se utilizan para definición de conceptos. < / d d >
< d t > Listas desordenas< / d t >
< d d >Son listas de elementos que se encuentran antecedidas de una viñeta. < / d d >
< / d l >

Enlaces

En un texto el cual al seleccionarlo ejecuta una acción de direccionamiento a otra página web o a otro punto del mismo sitio web, descargar archivos que se encuentra alojados en el mismo servidor de la página web.
Para la creación de enlaces utilizamos la siguiente etiqueta:
< a href = » https://www.unad.edu.co/»>Ir a Unad < / a >
El resultado será Ir a la Unad

Parámetros y tipos de enlaces

Se pueden utilizar parámetros en la etiqueta los cuales tendrán una funcionalidad diferentes para el enlace.
Href : Enlaza a documentos html internos o externos
Mailto: Enlaza con una dirección de correo electrónico
File: Enlace con un archivo
Name: Se utiliza para enlazar a través de los contenidos de la misma página
